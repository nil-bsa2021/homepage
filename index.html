Where the world builds software

Millions of developers and companies build, ship, and maintain their software on GitHub—the largest and most advanced development platform in the world.

201402-DecCensorshipRoscomnadzor, Russia's regulatory agency, blocks GitHub for hosting various copies of a suicide manual. Because GitHub uses HTTPS, which encrypts data between a user's computer and GitHub, internet service providers (ISP) are forced to block the whole website instead of the pages in question. Complying ISPs included Beeline, MTS, MGTS, and Megafon. Maxim Ksenzov, the Deputy Head of the Roscomnadzor, said in a statement that the block was due to GitHub not complying with earlier takedown requests for the manual on October 10, 2014. GitHub was also momentarily blocked on October 2, 2014 until the original copy of the manual was deleted.
